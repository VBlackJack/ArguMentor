## Quick Start âš¡

### Step 1: Create your first topic (30 seconds)

1. Tap the **+** button at bottom right
2. Give it a **title** (e.g., "Benefits of daily reading")
3. Add a **summary** for context
4. Choose your **posture** (neutral, critical, skeptical, opponent, comparative)
5. Tap **Save**

![Create a topic](/docs/images/screenshot-create-topic-en.png)

### Step 2: Add a claim (1 minute)

1. Open your topic
2. Tap **"Add claim"**
3. Formulate your argument (e.g., "Daily reading improves concentration")
4. Choose: **For**, **Against** or **Neutral**
5. Evaluate **strength**: weak, medium or strong
6. **Tip**: Use the ğŸ¤ button to dictate!

### Step 3: Document with evidence (1 minute)

1. From your claim, tap **"Add evidence"**
2. Choose the **type**: quote, statistic, scientific study, testimony, example
3. Add the evidence **content**
4. (Optional) Link it to a bibliographic **source**
5. Evaluate its **quality**: low, medium or high

### Complete Example

**Topic**: "Benefits of daily reading"
**Claim (For)**: "Daily reading improves concentration"
**Evidence (Study)**: "Stanford University study (2023): 30 minutes of reading per day increases attention span by 40% after 6 months"
**Source**: Journal of Cognitive Enhancement, 2023

**There you go, you've created your first documented argument! ğŸ‰**

#### PDF or Markdown Export (one topic)

4. Select where to save the document

### Import Data

#### JSON Import

1. Menu **â˜°** â†’ **Import/Export**
2. Tap **"Import"**
3. Adjust the **duplicate detection threshold** (85%-95%)
   - **85%**: Permissive (detects more duplicates)
   - **90%**: Balanced (recommended)
   - **95%**: Strict (detects fewer duplicates)
4. Select your JSON file
5. **Preview** changes (new items, updates, detected duplicates)
6. Confirm import

**Note**: Import is smart and automatically avoids duplicates.

### JSON Editor on PC

ArguMentor includes an HTML5 editor to create JSON files on computer:

1. Open the `json-editor.html` file in your browser
2. Use the **Visual Editor** (graphical interface, no code needed)
3. Create your topics, arguments and evidence visually
4. Export the JSON file
5. Import it into the mobile app

**Ideal for**: Creating argument corpora on PC, preparing educational exercises, collaborating with a team.

### App won't install

1. Verify you've allowed installation from unknown sources (Settings â†’ Security)

### Voice dictation doesn't work

**Problem**: The ğŸ¤ button doesn't respond or shows an error

**Solutions**:
1. Check you've granted microphone permission (Android Settings â†’ Apps â†’ ArguMentor â†’ Permissions)
2. Ensure your microphone works (test with another app)
3. Verify the language selected in ArguMentor matches the language you're speaking
4. Restart the app

---

### JSON import fails

**Problem**: Error message during import

**Solutions**:
1. Verify your JSON file is valid:
   - Open it with a text editor
   - Check it contains `"schemaVersion": "1.0"`
   - Use an online JSON validator (jsonlint.com)
2. If the file is very large (>50 MB), try splitting it into smaller files
3. Consult the [detailed JSON guide](IMPORT_JSON_GUIDE.md)

---

### App is slow

**Problem**: Slowdowns during use

**Solutions**:
1. If you have lots of data (>1,000 topics), search may be slower
2. Export your old analyses to JSON and archive them
3. Restart the app
4. Free up memory on your device (close other apps)

---

### PDF export doesn't generate

**Problem**: Error during PDF export

**Solutions**:
1. Check you have enough available storage space
2. Ensure you've chosen a valid save location
3. If the topic is very large (>100 arguments), export may take a few seconds
4. Try Markdown export instead, then convert to PDF on PC

---

### Can't find my data after an update

**Problem**: Data disappeared after installing a new version

**Solution**:
1. Data should be preserved automatically
2. If not, restore your latest JSON backup (Menu â†’ Import/Export â†’ Import)
3. **Tip**: Always export your data before a major update
